# docs/

## 目錄索引

| 目錄 | 用途 |
|------|------|
| `dev/` | 系統技術文件：架構說明、API 參考、開發指引、ADR |
| `design/` | 產品設計檔案（`.pen` Pencil 設計稿） |
| `_external/` | 外部參考資料：比賽文件、競品分析、舊原型（不描述系統行為） |

## 快速入口

- **AI Session 啟動點**：`dev/architecture.md` — 系統全貌，一份文件看懂架構
- **架構決策紀錄**：`dev/decisions/` — 每個重要技術選擇的 Why
- **API 文件**：`dev/elevenlabs-api-reference.md`、`dev/fal-ai-api-reference.md`
- **交接文件**：`dev/handoffs/`
