### Q13 — Brief Product Description (English)
```
Moltos is the first AI companion that detects your stress before you say a word. By integrating with messaging platforms (LINE, Telegram, WhatsApp, Gmail, Slack), Moltos passively monitors behavioral patterns — message volume trends, reply latency changes, late-night activity spikes, and unread message pileup — to compute a real-time Calm Index (0-100) using our open-source algorithm (MIT License). When the index drops below a personal threshold, Moltos proactively initiates a voice call with real-time emotion analysis (speech rate, pitch, pause patterns) to provide empathetic AI support. Unlike existing mental health chatbots that rely on self-reporting, Moltos detects distress from real behavioral data. Privacy-first: only metadata is extracted, conversations are end-to-end encrypted, and all computation runs on-device.
```
